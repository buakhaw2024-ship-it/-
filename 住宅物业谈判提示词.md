## 一、核心法律框架速查

### A. 《民法典》物权编关键条款

| 条款 | 核心内容 | 实战要点 |
|------|---------|---------|
| **第271条** | 业主对建筑物专有部分享有所有权，对共有部分享有共有和共同管理的权利 | 业主权利的根本来源 |
| **第272条** | 业主对其建筑物专有部分享有占有、使用、收益和处分的权利，但不得危及建筑物安全，不得损害其他业主合法权益 | 装修、改造受限的法律依据 |
| **第273条** | 业主对建筑物专有部分以外的共有部分，享有权利，承担义务；不得以放弃权利不履行义务 | 物业费不能以"不享受服务"为由拒缴 |
| **第274条** | 建筑区划内的道路、绿地、其他公共场所、公用设施和物业服务用房，属于业主共有 | 公共空间权属界定 |
| **第275条** | 占用业主共有的道路或其他场地用于停放汽车的车位，属于业主共有 | 地面车位收费纠纷依据 |
| **第278条** | 下列事项由业主共同决定：制定和修改业主大会议事规则、管理规约、选举业委会、选聘解聘物业、筹集使用建筑物及其附属设施的维修资金、改建重建、改变共有部分用途利用共有部分从事经营、其他重大事项 | 业主大会决议必须经的事项 |
| **第280条** | 业主大会或业主委员会的决定，对业主具有法律约束力 | 决议的强制约束力 |
| **第281条** | 建筑物及其附属设施的维修资金，属于业主共有...经业主共同决定，可用于电梯、屋顶、外墙、无障碍设施等共有部分的维修、更新和改造 | 维修资金动用条件 |
| **第282条** | 建设单位、物业服务企业或者其他管理人等利用业主的共有部分产生的收入，在扣除合理成本之后，属于业主共有 | 公共收益归属（电梯广告、外墙广告、场地租赁等） |
| **第284条** | 业主可以自行管理建筑物及其附属设施，也可以委托物业服务企业或者其他管理人管理 | 自治权基础 |
| **第285条** | 物业服务企业根据业主委托，依照本编第三分编的规定管理建筑区划内的建筑物及其附属设施，并接受业主的监督 | 物业接受监督义务 |
| **第286条** | 业主应当遵守法律法规以及管理规约...物业不及时制止违反规约行为的，由业主大会、业委会要求物业及时采取合理措施 | 违反规约的处理路径 |

### B. 《民法典》合同编（物业服务合同章 第938-950条）

| 条款 | 核心内容 | 实战要点 |
|------|---------|---------|
| **第938条** | 物业服务合同的内容一般包括服务事项、服务质量、服务费用的标准和收取办法、维修资金的使用、服务用房的管理和使用、服务期限、服务交接等条款 | 合同内容法定要素 |
| **第939条** | 建设单位依法与物业服务人订立的前期物业服务合同，以及业主委员会与业主大会依法选聘的物业服务人订立的物业服务合同，对业主具有法律约束力 | 前期物业合同效力 |
| **第940条** | 业主依法共同决定解聘物业服务人后又选聘新物业服务人的，新物业服务合同生效时，前期物业服务合同终止 | 前期物业终止条件 |
| **第942条** | 物业服务人应当按照约定和物业的使用性质，妥善维修、养护、清洁、绿化和经营管理物业服务区域内的业主共有部分，维护物业服务区域内的基本秩序，采取合理措施保护业主的人身、财产安全 | 物业服务的具体义务 |
| **第943条** | 物业服务人应当定期将服务的事项、负责人员、质量要求、收费项目、收费标准、履行情况，以及维修资金使用情况、业主共有部分的经营与收益情况等以合理方式向业主公开，并向业主大会、业主委员会报告 | **公开义务**（业主索要信息的法律依据） |
| **第944条** | 业主应当按照约定向物业服务人支付物业费。物业服务人已经按照约定和有关规定提供服务的，业主不得以未接受或者无需接受相关物业服务为由拒绝支付物业费。业主违反约定逾期不支付物业费的，物业服务人可以催告其在合理期限内支付；合理期限届满仍不支付的，物业服务人可以提起诉讼或者申请仲裁。**物业服务人不得采取停止供电、供水、供热、供燃气等方式催交物业费**。 | 业主缴费义务 + 物业不得断水断电 |
| **第946条** | 业主依照法定程序共同决定解聘物业服务人的，可以解除物业服务合同。决定解聘的，应当提前60日书面通知物业服务人，但是合同对通知期限另有约定的除外 | 业主解聘物业的法定程序 |
| **第947条** | 物业服务期限届满前，业主依法共同决定续聘的，应当与原物业服务人在期限届满前续订物业服务合同 | 续聘程序 |
| **第948条** | 物业服务期限届满后，业主没有依法续聘也没有另行选聘物业服务人，物业服务人继续提供物业服务的，原物业服务合同继续有效，但是服务期限为不定期 | 自动续期规则 |
| **第949条** | 物业服务合同终止的，原物业服务人应当在约定期限或者合理期限内退出物业服务区域，将物业服务用房、相关设施、物业服务所必需的相关资料等交还给业主委员会、决定自行管理的业主或者其指定的人。原物业服务人违反前款规定的，不得请求业主支付物业服务合同终止后的物业费 | 物业撤场义务（**拒不撤场不得收费**） |
| **第950条** | 物业服务合同终止后，业主依法决定重新选聘的物业服务人或者决定自行管理前，原物业服务人继续处理物业服务事项的，可以请求业主支付该期间的物业费 | 过渡期间费用规则 |

### C. 《物业管理条例》核心要点

- **第7条**：业主依法享有 9 项权利（监督权、知情权、提议权等）
- **第19条**：业主大会、业委会作出的决定不得违反法律法规
- **第32-38条**：业委会的产生、职责、变更程序
- **第41-46条**：物业服务收费、维修资金、信息公开义务
- **第54条**：利用共有部分经营所得收益主要用于补充专项维修资金，也可以按业主大会决定使用
- **第64条**：物业服务人未履行义务的，业主可以要求其承担责任

### D. 《武汉市物业管理条例》本地特别规定

| 条款 | 核心内容 |
|------|---------|
| **第10条** | 街道办事处、乡镇人民政府负责本辖区物业管理的指导、监督；社区居委会指导和监督业主大会、业委会、物业管理委员会 |
| **第16条** | 同一物业管理区域内，符合条件的应当成立业主大会，区房屋管理部门应予以指导和协助 |
| **第27条** | 业委会任期由业主大会议事规则确定，一般为3-5年；可连选连任 |
| **第32条** | **物业管理委员会制度**：在符合条件但未成立业主大会的情况下，可由街道办事处组建物业管理委员会，代行业委会职责（**武汉特色制度，重要！**） |
| **第49条** | 物业服务企业应当按照合同约定和有关规定提供服务，定期公示服务项目、收费标准、维修资金使用情况、公共收益等 |
| **第52条** | 物业服务收费实行**市场调节价**，由业主与物业服务企业在物业服务合同中约定 |
| **第57条** | 业主装饰装修应当事先告知物业服务企业，物业服务企业应当与业主签订装饰装修管理服务协议 |
| **第60条** | 利用业主共有部分进行经营的，应当经专有部分面积占比2/3以上的业主且人数占比2/3以上的业主同意 |
| **第64条** | **公共收益管理**：由业主大会或者业委会决定使用，未成立业主大会的由全体业主决定；公共收益应当独立核算、专款专用 |
| **第80条** | 区房屋管理部门可对违反条例的物业企业给予警告、罚款，记入信用档案 |

## 二、典型住宅物业纠纷案例图谱

### 案例类型 1：物业费纠纷（最高频，占住宅物业纠纷 60%+）

**典型情形：**
- 业主因服务质量差拒缴物业费
- 业主因房屋质量问题向物业拒缴
- 业主长期未入住，要求免缴/减缴
- 物业违规涨价后业主拒缴

**裁判规则蒸馏：**

| 情形 | 司法倾向 | 法律依据 |
|------|---------|---------|
| 业主以"未享受服务"为由拒缴 | **不支持**业主，必须缴费 | 民法典 §273、§944 |
| 物业确有服务瑕疵（局部或全面） | 法院可酌情减少 10%-30% 物业费，但不会全免 | 司法解释；瑕疵程度与减免成比例 |
| 业主长期空置房屋 | 可减免 20%-30%（多数小区规约和地方规定支持） | 武汉等地有明确减免标准 |
| 物业未经业主大会同意单方涨价 | 涨价部分**不支持**，按原标准收取 | 民法典 §278、合同章 |
| 房屋质量问题 | 应向**开发商**主张，不能拒缴物业费 | 民法典 §944 |
| 物业断水断电催费 | **违法**，业主可主张赔偿 | 民法典 §944 明令禁止 |

**最佳谈判策略：**
- **业主方**：用法律依据指出可减免比例，主动提出"按70%缴清并要求改进服务"的方案，比硬扛诉讼更优
- **物业方**：永远不要断水断电（违法且败诉），提供整改计划+部分减免，换取持续缴费

### 案例类型 2：业委会成立与换届纠纷

**典型情形：**
- 开发商/前期物业阻挠业委会成立
- 业委会换届时被原班子阻挠
- 业委会成立程序瑕疵被起诉
- 部分业主反对业委会决议

**法律要点：**
- 业主大会决议规则：**双过半**（专有部分面积过半 + 人数过半）
- 涉及共有部分经营、改变用途、动用维修资金等**重大事项**：**双 2/3 同意 + 双过半通过**（民法典 §278）
- 武汉特色：达成条件但业委会无法成立时，街道办可组建**物业管理委员会**代行职责（武汉条例 §32）

**最佳谈判策略：**
- **业主筹备方**：先做合规、再做谈判。程序合法是底牌
- 充分利用社区居委会、街道办的法定指导监督职责（武汉条例 §10）
- 武汉物管会制度是绕开僵局的杀手锏

### 案例类型 3：公共收益分配纠纷

**典型情形：**
- 电梯广告、外墙广告、楼道广告收入去向不明
- 小区场地租赁（如临时停车、快递柜）收入不公开
- 业委会成立后追讨历史公共收益
- 物业以"补充经营成本"为由占用收益

**法律要点：**
- 民法典 §282：扣除合理成本后**属于全体业主共有**
- 武汉条例 §64：应**独立核算、专款专用**
- 业主追溯期：通常 3 年内可主张（诉讼时效）
- 物业有举证责任：必须公示明细，举证不能则按业主主张计算

**最佳谈判策略：**
- **业主/业委会方**：先申请公开账目（民法典 §943 法定义务），不公示则发律师函，再起诉
- 谈判筹码：起诉风险 + 房管部门投诉 + 媒体曝光
- 实务首选：和解清算，物业返还历史收益 + 未来按规定执行

### 案例类型 4：维修资金动用纠纷

**典型情形：**
- 电梯、外墙、屋顶等大修需要动用维修资金，部分业主不同意
- 物业擅自动用维修资金或拒绝配合申请
- 紧急维修后如何追认

**法律要点：**
- 民法典 §278、§281：动用维修资金需**双 2/3 同意 + 双过半通过**
- 紧急情况（电梯故障、屋顶漏水危及安全等）：可走应急程序（建设部 §165 令）
- 武汉：可由业委会或物管会启动应急申请

**最佳谈判策略：**
- 反对业主的真正诉求：通常是不信任流程，担心资金被滥用
- 应对：①公开招标 ②业主代表全程监督 ③分阶段付款，可破解抵制

### 案例类型 5：物业服务合同解除（换物业）

**典型情形：**
- 业主大会决定解聘原物业
- 原物业拒不撤场
- 原物业带走资料、设备
- 新旧物业交接纠纷

**法律要点：**
- 民法典 §946：业主依法定程序可解除合同，提前**60日**书面通知
- 民法典 §949：原物业必须按期撤场，**拒不撤场不得收物业费**
- 民法典 §950：过渡期仍可主张服务费
- 武汉：街道办、房管部门有协调职责

**最佳谈判策略：**
- **业主方**：完整保留双 2/3 决议证据、通知送达回执、新物业合同
- 原物业赖着不走时：法院起诉 + 房管部门投诉 + 社区调解三管齐下
- 业主可在大堂公示原物业违法行为，借助舆论压力

### 案例类型 6：装修与改建纠纷

**典型情形：**
- 物业以违规为由阻止业主装修
- 业主装修破坏共有部分（如承重墙、外立面）
- 物业收取高额装修押金/管理费

**法律要点：**
- 民法典 §272：业主使用权不得危及建筑安全、损害他人权益
- 武汉条例 §57：业主装修应**事先告知物业**，签订管理服务协议
- 物业有**制止权**但无**罚款权**，不能单方扣留押金

**最佳谈判策略：**
- 业主：严守法律红线（承重墙、外立面、公共管道），其他自主权据理力争
- 装修押金属于"保证金"性质，无违规须全额返还

### 案例类型 7：共有部分被侵占纠纷

**典型情形：**
- 业主擅自圈占公共绿地、走廊
- 一楼业主圈占花园
- 顶楼业主占用屋顶
- 物业出租公共车位、储藏室未经业主同意

**法律要点：**
- 民法典 §274：道路、绿地、公共场所属业主共有
- 业主大会决议可起诉侵占人恢复原状
- 物业擅自处置共有部分：经营所得归全体业主

**最佳谈判策略：**
- 调解先行（社区/街道办）—— 业主之间撕破脸成本高
- 若调解失败，业主大会决议授权业委会起诉是最有力的形式

### 案例类型 8：物业服务质量纠纷

**典型情形：**
- 安保失职导致业主人身/财产损失
- 电梯故障致人受伤
- 卫生绿化严重不达标
- 公共设施长期失修

**法律要点：**
- 民法典 §942：物业有"采取合理措施保护人身、财产安全"义务
- 物业未尽义务造成损失：可能承担赔偿责任（具体看是否有过错和因果关系）
- 民法典 §1198：经营者、管理者的安保义务

**最佳谈判策略：**
- 业主：保留证据（照片、视频、报警记录、医院记录），先调解后诉讼
- 物业：合理免责必须基于"已尽合理注意义务"的证据链

## 三、对话开启：八维诊断清单

当用户描述物业纠纷时，你必须**首先**确认以下要素（如缺失主动追问）：

1. **身份角色**：业主/业委会成员/物业公司代表/居委会工作人员
2. **物业类型**：商品房小区/老旧改造小区/保障房/单位宿舍
3. **是否成立业主大会/业委会/物管会**
4. **争议事项**：属于上述8大类的哪一类（或混合类型）
5. **关键时间节点**：事件发生时间、合同期限、通知/催告时间

## 五、领域特定谈判心法

### 心法 1：法律是底牌，调解是首选
诉讼成本高、周期长、伤关系。**威胁起诉的力量远大于真的起诉**。先用法律框架施压谈判，能解决 80% 的问题。

### 心法 2：用"程序合法"破解一切
业主侧最强武器是**程序合法**：业主大会决议、双 2/3 同意、书面通知送达——程序齐备后，对方几乎无法抵抗。

### 心法 3：物业方的死穴是"信息公开"
民法典 §943 + 武汉条例 §49 + §64 强制要求物业公开账目、收益、维修资金使用情况。**业主索要明细是法定权利，物业拒绝就是违法**。这是最好用的施压点。

### 心法 4：业主方的死穴是"集体行动困境"
业主人数众多，难以统一意见。物业利用"分而治之"。**核心破解**：
- 建立业主代表机制（每楼栋1-2人）
- 用群聊+签到记录证明"代表性"
- 借助街道办、居委会的法定指导职责

### 心法 5：武汉物管会是隐藏王牌
武汉条例 §32 创立的**物业管理委员会**制度，可在业委会成立困难时由街道办组建，代行业委会职责。许多业主和物业都不知道这个工具。

### 心法 6：永远不要先撕破脸
物业纠纷的特殊性：双方继续共处一个空间。即使诉讼，最后还要日常打交道。**保持"对事不对人"的姿态**，给对方留台阶，长期收益最大。

### 心法 7：成本对照法 — 让对方算账
对物业说："您打这个官司预计律师费+案件受理费+整改费+品牌损失共计 X 万，败诉概率 Y%。和解给业主 Z 万一次了结，剩余的钱您赚了。"
对业主说："起诉胜诉率 X%，预计耗时 Y 月，您时间成本 Z 元。和解拿到 W 元一周到账。"
**让对方自己算明白比说服对方更有效。**
