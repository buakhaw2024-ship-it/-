# 全谱系谈判博弈模拟系统 v5.9

> 一个离线可用的谈判训练游戏——双击 HTML 文件，浏览器直接打开，无需安装、无需联网。

---

## 功能一览

| 模块 | 说明 |
|------|------|
| **传奇训练中心**（主体验） | 8 条历史剧情链 + 12 位导师电影化试炼场 + 人物精通档案 + 本地回放档案馆 |
| **历史剧情链** | 孙子伐楚 / 张仪连横 / 慈禧垂帘 / 芒格并购 / 特朗普地产 / 默克尔欧债 / 撒切尔财政 / 三巨头雅尔塔——34 幕 × 4 回合 × 5 个时代化话术选项 |
| **电影化试炼场** | 12 位导师 × 6 幕剧本，导演幕牌、五方压力、4 条局势条、对手记忆系统、三层复盘 |
| **场景实战训练** | 5 大经典博弈 × 6 对手 = 30 个实战情境包：每个选项都是一句真实话术 |
| **谈判弧线模式** | 7 条主题剧情弧线 × 5 阶段决策链 × 多分支结局 |
| **人物精通档案** | 22 位导师档案：核心能力 + 核心陷阱 + 三档等级 + 5 项能力 + 行为徽章 + 高频风险 + 最近复盘 |
| **技巧熟练度（螺旋成长）** | 8 大核心谈判技巧：入门→熟练→精通→大师，达"熟练"解锁高压模式 |
| **可解释复盘** | "你选了 X / 最佳是 Y / 差距 N 分" 逐节对比 + 基于真实回合数据的改进建议 |
| **策略理论库** | 6 大宗师方法论 + 8 项技巧档案 + 训练处方 |
| **导师推荐系统** | 每局结束后根据对手类型推荐克制宗师 |
| **存档导出/导入** | 一键导出 JSON 备份，跨浏览器/设备恢复训练进度 |
