# 帝斯曼财务仪表盘 · 局域网共享（自动同步）

把这个仪表盘放在**一台电脑（主机）**上运行，**同一个 WiFi/局域网**里的同事用手机或电脑浏览器就能直接打开查看；**你一更新，大家的页面会自动刷新**，不用手动按 F5。

---

## 一、准备（只做一次）

主机电脑需要装 **Python 3**：
- **Windows**：到 https://www.python.org/downloads/ 下载安装，安装时**务必勾选「Add Python to PATH」**。
- **macOS**：系统一般自带 `python3`；若没有，同样到上面网址下载。

> 图表库（chart.umd.js）已经放在本文件夹里，**断网也能正常显示**，无需额外联网。

## 二、启动共享

把**整个文件夹**拷到主机电脑，然后：
- **Windows**：双击 `启动-Windows.bat`
- **macOS**：双击 `启动-Mac-Linux.command`
  （若提示无法打开：右键 → 打开；或终端执行 `bash 启动-Mac-Linux.command`）

启动后，黑色窗口会显示类似下面的网址，**并直接画出一个二维码**：

```
 本机打开：      http://localhost:8000
 同一 WiFi 的其他设备（手机/电脑）打开：
      http://192.168.1.23:8000      <-- 把这个发给同事
 手机扫码直接打开（需连同一个 WiFi）：http://192.168.1.23:8000

   █▀▀▀▀▀█  ▀▄ █ █▀▀▀▀▀█
   █ ███ █ ▀█▀▄▀ █ ███ █      ← 窗口里会显示真正的二维码
   █ ▀▀▀ █ █ ▀▀█ █ ▀▀▀ █        （手机相机/微信扫一扫即可）
   ▀▀▀▀▀▀▀ █▄▀▄█ ▀▀▀▀▀▀▀
   ...（此处为示意，实际为可扫描的二维码）...
```

## 三、同事如何查看（两种方式）

**方式 A：扫码（最快）**
- 用手机相机或微信「扫一扫」，对准黑窗口里的二维码，点弹出的链接即可打开。

**方式 B：输入网址**
- 在浏览器地址栏输入主机窗口里显示的网址（例如 `http://192.168.1.23:8000`）。

> 两种方式都要求：同事的手机/电脑连的是**同一个 WiFi/局域网**。
> 二维码里装的就是那条 `192.168.x.x` 局域网地址，所以用 4G/流量或别的 WiFi 扫，能识别但打不开。
> 二维码是本机离线生成的，不需要联网。

## 四、更新数据（你改，大家自动同步）

把**新的仪表盘文件**覆盖本文件夹里的 `index.html`（文件名保持 `index.html`）。
保存后，**所有正在看页面的人会在 1～2 秒内自动刷新**为最新内容。

- 不用重启程序，也不用通知大家刷新。
- 同理，替换 `chart.umd.js` 等其他文件也会触发自动刷新。

## 五、关闭共享

在黑色窗口里按 **Ctrl + C**，或直接关闭该窗口。

## 常见问题

- **同事打不开 / 转圈**
  - 确认在**同一个 WiFi**（公司访客网络、手机热点常常互相隔离，打不通）。
  - 主机第一次启动时，**防火墙弹窗请选「允许」**（Windows 选「专用网络」）。
  - 用主机窗口里显示的 `192.168.x.x` 网址，不要用 `localhost`（那只有主机自己能开）。

- **想换端口**（默认 8000，被占用时）
  - Windows：先在命令行 `set PORT=8080`，再运行 `python serve.py`。
  - macOS/Linux：`PORT=8080 python3 serve.py`。

- **主机要一直开着吗？**
  - 是的。共享期间主机程序需保持运行、**电脑不要休眠**；关掉窗口或休眠后大家就访问不了。

- **安全说明**
  - 这是**只读展示**，访问者只能看，不能改你的数据。
  - 仅在局域网内可见；如需公网访问请另行配置（不建议直接暴露到公网）。
