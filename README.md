# 全谱系谈判博弈模拟系统 v5.2

> 一个离线可用的谈判训练游戏——双击 HTML 文件，浏览器直接打开，无需安装、无需联网。

---

## 功能一览

| 模块 | 说明 |
|------|------|
| **谈判博弈对局** | 7 类 AI 对手（理性型、强硬型、情感型、操控型等），每局随机策略组合 |
| **叙事化实战场景** | 5 大经典博弈 × 6 对手 = 30 个实战情境包：证监会核查、遗产分配、反垄断约谈、大客户压价…… 每个选项都是一句真实话术 |
| **传奇导师系列** | 6 位 SSR 宗师（道森、费舍尔、沃斯、科恩、戴蒙德、谢尔），22 张官方肖像 |
| **导师推荐系统** | 每局结束后根据对手类型自动推荐克制宗师，一键跳转详情 |
| **技巧熟练度（螺旋成长）** | 8 大核心谈判技巧档案（同理心/校准问题/BATNA……），最佳选择累计升档：入门→熟练→精通→大师 |
| **高压模式** | 技巧达熟练后解锁：错误代价 ×1.5，逼你在压力下稳定输出 |
| **可解释复盘** | "你选了 X / 最佳是 Y / 差距 N 分" 逐节对比 + 基于真实回合数据的改进建议 |
| **策略图鉴** | 全宗师卡面展示，含战术难度星级、克制目标、经典语录 |
| **传奇谈判者名人堂** | 22 位 UR/LR 历史名人（克里奥帕特拉、丘吉尔、撒切尔……），含稀有度边框 |
| **技能卡系统** | 盲盒抽卡 + 稀有度筛选 + 卡组配置，博弈中触发技能特效 |
| **收藏 & 成就** | 段位徽章、历史战绩、剧情弧线 |
